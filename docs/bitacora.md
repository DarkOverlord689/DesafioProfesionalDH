# 📑 Bitácora de Proyecto - Sprint 1 (Definición Técnica Final)

## 🎯 1. Definición del Proyecto
**Nombre de la Aplicación:** Darksishop
**Propósito:** Plataforma integral de reserva de alojamientos y experiencias, diseñada con las últimas tecnologías para garantizar velocidad y escalabilidad.

## 🛠️ 2. Stack Tecnológico (Actualizado)
### **Frontend (Interfaz de Usuario)**
* **Framework:** React.js (v18+) con Vite.
* **Enrutamiento:** React Router Dom (v6+).
* **Estilos:** CSS3 Puro siguiendo la identidad visual del desafío.

### **Backend (Motor y Lógica)**
* **Lenguaje:** Java 21 (LTS).
* **Framework:** Spring Boot 4.0.5.
* **Gestión de Dependencias:** Maven.

### **Persistencia**
* **Motor:** MySQL 8.0.
* **ORM:** Spring Data JPA.
